# Платформа для геймдев-разработчиков

### Используемые технологии (предварительно)
JDK: OpenJDK 25
IDE (java): IntelliJ Idea
Плагин для IDE: Explyt Spring
СУБД: MySQL 
HTTP клиент: Postman

### Область применения проекта
ShowMyGame — это веб-платформа, где независимые разработчики, студии и издатели могут делиться демо-версиями игр, искать партнеров и находить работу или команду.  
Пользователь видит игру в ленте и сразу может запустить и оценить демо в браузере.
Платформа сочетает в себе элементы:
- игрового портфолио;
- профессиональной сети для разработчиков;
- площадки для тестирования и подбора талантов.

### Участники системы
| Роль                        | Описание                                                                                      |
| --------------------------- | --------------------------------------------------------------------------------------------- |
| **Разработчик**             | Загружает демо-версии своих игр, ведет профиль, получает отзывы и ищет издателей или команду. |
| **Студия**                  | Публикует проекты от имени студии, объединяет несколько разработчиков в команду.              |
| **Игрок / Пользователь**    | Просматривает и запускает демо прямо в браузере, ставит оценки и оставляет комментарии.       |
| **Издатель / Работодатель** | Изучает проекты и профили разработчиков, ищет перспективные игры и команды.                   |

### Основные сценарии использования
#### Разработчик
1. Регистрируется на платформе и создает профиль.  
2. Загружает демо-версию игры (WebGL, HTML5).  
3. Добавляет описание, теги, жанр, обложку и билд.  
4. Публикует проект — он появляется в общей ленте.  
5. Обновляет билд или редактирует описание при необходимости.
####  Студия
1. Создает студийный профиль. 
2. Добавляет участников (разработчиков) в команду.  
3. Публикует игры от имени студии.  
4. Просматривает статистику по проектам и взаимодействует с издателями.
####  Игрок
1. Просматривает ленту проектов.
2. Фильтрует игры по тегам и жанрам.
3. Нажимает на карточку — запускает билд игры в браузере.
4. Ставит оценку и оставляет комментарий.
####  Издатель
1. Просматривает ленту популярных игр.  
2. Использует фильтры по жанрам и рейтингу.  
3. Находит интересный проект и связывается с автором.  
4. Отправляет приглашение на сотрудничество.

### Сущности
#### 1. `user`
| Поле          | Тип                                         | Описание                              |
| ------------- | ------------------------------------------- | ------------------------------------- |
| `id`          | integer                                     | Уникальный идентификатор пользователя |
| `username`    | string                                      | Логин                                 |
| `email`       | string                                      | Почта                                 |
| `role`        | enum (developer, studio, publisher, player) | Роль пользователя                     |
| `description` | text                                        | Описание профиля                      |
| `avatar`      | image                                       | Аватар                                |
| `created_at`  | date                                        | Дата регистрации                      |
#### 2. `project`
| Поле          | Тип     | Описание                      |
| ------------- | ------- | ----------------------------- |
| `id`          | integer | Уникальный идентификатор игры |
| `author_id`   | integer | Автор или студия              |
| `title`       | string  | Название проекта              |
| `description` | text    | Описание                      |
| `tags`        | string  | Список тегов                  |
| `genre`       | string  | Жанр                          |
| `build`       | ?       | Исполняемый билд              |
| `title`       | image   | Обложка                       |
| `created_at`  | date    | Дата публикации               |
#### 3. `rate`
| Поле         | Тип                 | Описание                         |
| ------------ | ------------------- | -------------------------------- |
| `id`         | integer             | Уникальный идентификатор оценки  |
| `project_id` | integer             | Игра, к которой относится оценка |
| `user_id`    | integer             | Автор оценки                     |
| `rate`       | integer (от 1 до 5) | Оценка                           |
| `created_at` | date                | Время выставления оценки         |
#### 4. `comment`
| Поле         | Тип     | Описание                              |
| ------------ | ------- | ------------------------------------- |
| `id`         | integer | Уникальный идентификатор комментария  |
| `project_id` | integer | Игра, к которой относится комментарий |
| `user_id`    | integer | Автор комментария                     |
| `text`       | text    | Текст комментария                     |
| `created_at` | date    | Время публикации                      |
#### 5. `message`
| Поле          | Тип     | Описание                |
| ------------- | ------- | ----------------------- |
| `id`          | integer | Уникальный ID сообщения |
| `sender_id`   | integer | Отправитель             |
| `receiver_id` | integer | Получатель              |
| `content`     | text    | Текст сообщения         |
| `sent_at`     | time    | Время отправки          |
| `is_read`     | boolean | Прочитано или нет       |

### Связи между таблицами
| Связь                                   | Тип | Направление                                    |
| --------------------------------------- | --- | ---------------------------------------------- |
| Пользователь → Проекты                  | 1:N | Один пользователь имеет много проектов         |
| Проект → Комментарии                    | 1:N | Один проект имеет много комментариев           |
| Пользователь → Комментарии              | 1:N | Один пользователь пишет много комментариев     |
| Пользователь ↔ Пользователь (Сообщения) | M:N | Пользователи могут обмениваться сообщениями    |
| Студия → Пользователи                   | 1:N | Студия имеет несколько участников              |
