# ShowMyGame — Платформа для геймдев-разработчиков

## О проекте

**ShowMyGame** — это инновационная веб-платформа, созданная для объединения игровых разработчиков, издателей и игроков. Платформа позволяет разработчикам публиковать демо-версии игр, которые можно запускать напрямую в браузере без установки, формируя интерактивное портфолио. Издатели и работодатели получают возможность для поиска талантов и перспективных проектов, а игроки — доступ к уникальному контенту и возможность влиять на развитие инди-игр.

### Основные возможности:
- **Мгновенный запуск игр** в браузере через WebGL/HTML5
- **Интерактивное портфолио** для разработчиков
- **Система оценок и комментариев** для обратной связи
- **Поиск и фильтрация** проектов по жанрам, тегам и рейтингам
- **Персональные закладки** для сохранения понравившихся проектов
- **Управление профилями** с поддержкой разных ролей

## Технологический стек

### Бэкенд
- **Java** — основной язык программирования
- **Spring Boot** — фреймворк для создания веб-приложений
- **MySQL** — система управления базами данных

### Фронтенд

### Инструменты разработки
- **IntelliJ IDEA** — интегрированная среда разработки
- **DBeaver** — управление базой данных
- **Postman** — тестирование API
- **Git** — система контроля версий


## Роли пользователей

| Роль            | Описание                            | Основные возможности                                                |
| --------------- | ----------------------------------- | ------------------------------------------------------------------- |
| **Игрок**       | Обычный пользователь платформы      | Просмотр проектов, запуск игр, комментирование, оценка              |
| **Разработчик** | Создатель игровых проектов          | Публикация проектов, управление портфолио, получение обратной связи |
| **Издатель**    | Представитель издательской компании | Поиск проектов, оценка потенциала, контакты с разработчиками        |
## Функциональные возможности
### Для всех пользователей:
- Регистрация и создание профиля
- Просмотр каталога проектов
- Запуск игр прямо в браузере
- Система комментариев
- Пятизвездочная система оценок
- Добавление проектов в избранное
- Просмотр профилей пользователей
### Для разработчиков:
- Загрузка игровых проектов (ZIP-архивы с WebGL/HTML5 билдами)
- Управление метаданными проектов (название, описание, жанры, теги)
- Загрузка обложек проектов
- Редактирование и удаление проектов


## Структура проекта

```
ShowMyGame/
├── src/main/java/ru/app/ShowMyGame/
│   ├── controllers/          # Spring MVC контроллеры
│   ├── entities/             # JPA-сущности
│   ├── repositories/         # Репозитории для доступа к данным
│   ├── services/             # Сервисы с бизнес-логикой
│   ├── helpers/              # Вспомогательные классы
│   └── ShowMyGameApplication.java # Точка входа
├── src/main/resources/
│   ├── static/              # Статические ресурсы (CSS, JS, изображения)
│   ├── templates/           # Thymeleaf шаблоны
│   └── application.properties # Конфигурация приложения
├── upload/                  # Загружаемые файлы (создаётся автоматически)
│   ├── avatars/            # Аватары пользователей
│   ├── builds/             # Игровые билды
│   └── projects/           # Обложки проектов
└── pom.xml                 # Конфигурация Maven
```


## Основные таблицы:

### **Таблица `users` - Пользователи**

| Поле               | Тип          | Ограничения                 | Описание                                  |
| ------------------ | ------------ | --------------------------- | ----------------------------------------- |
| `id`               | INT          | PRIMARY KEY, AUTO_INCREMENT | Уникальный идентификатор пользователя     |
| `username`         | VARCHAR(255) | NOT NULL, UNIQUE            | Имя пользователя для отображения          |
| `email`            | VARCHAR(255) | NOT NULL, UNIQUE            | Электронная почта для входа и уведомлений |
| `password_hash`    | VARCHAR(255) | NOT NULL                    | Хеш пароля (BCrypt)                       |
| `role`             | ENUM         | NOT NULL                    | Роль пользователя в системе               |
| `description`      | TEXT         | -                           | Текстовое описание профиля                |
| `avatar_file_name` | VARCHAR(255) | -                           | Имя файла аватара в системе хранения      |
| `created_at`       | DATE         | NOT NULL                    | Дата регистрации пользователя             |
|                    |              |                             |                                           |

### **Таблица `projects` - Игровые проекты**

| Поле                | Тип          | Ограничения                 | Описание                           |
| ------------------- | ------------ | --------------------------- | ---------------------------------- |
| `id`                | INT          | PRIMARY KEY, AUTO_INCREMENT | Уникальный идентификатор проекта   |
| `author_id`         | INT          | FOREIGN KEY (`users.id`)    | Автор проекта (ссылка на users.id) |
| `title`             | VARCHAR(255) | NOT NULL                    | Название проекта                   |
| `description`       | TEXT         | -                           | Подробное описание проекта         |
| `tags`              | TEXT         | -                           | Сериализованный JSON массив тегов  |
| `genres`            | TEXT         | -                           | Сериализованная строка жанров      |
| `build_type`        | VARCHAR(50)  | -                           | Тип игрового билда                 |
| `build_folder_name` | VARCHAR(255) | -                           | Имя папки с распакованным билдом   |
| `image_file_name`   | VARCHAR(255) | -                           | Имя файла обложки проекта          |
| `created_at`        | DATE         | NOT NULL                    | Дата публикации проекта            |

### **Таблица `comments` - Комментарии**

| Поле         | Тип      | Ограничения                           | Описание                                |
| ------------ | -------- | ------------------------------------- | --------------------------------------- |
| `id`         | INT      | PRIMARY KEY, AUTO_INCREMENT           | Уникальный идентификатор комментария    |
| `project_id` | INT      | FOREIGN KEY (`projects.id`), NOT NULL | Проект, к которому оставлен комментарий |
| `user_id`    | INT      | FOREIGN KEY (`users.id`), NOT NULL    | Автор комментария                       |
| `text`       | TEXT     | NOT NULL                              | Текст комментария                       |
| `created_at` | DATETIME | NOT NULL                              | Дата и время создания комментария       |

### **Таблица `rates` - Оценки**

| Поле         | Тип      | Ограничения                           | Описание                         |
| ------------ | -------- | ------------------------------------- | -------------------------------- |
| `id`         | INT      | PRIMARY KEY, AUTO_INCREMENT           | Уникальный идентификатор оценки  |
| `project_id` | INT      | FOREIGN KEY (`projects.id`), NOT NULL | Оцениваемый проект               |
| `user_id`    | INT      | FOREIGN KEY (`users.id`), NOT NULL    | Пользователь, поставивший оценку |
| `rate`       | TINYINT  | NOT NULL, CHECK (0 <= rate <= 5)      | Значение оценки (0-5 звезд)      |
| `created_at` | DATETIME | NOT NULL                              | Дата и время оценки              |

### **Таблица `bookmarks` - Избранное**

| Поле         | Тип      | Ограничения                           | Описание                          |
| ------------ | -------- | ------------------------------------- | --------------------------------- |
| `id`         | INT      | PRIMARY KEY, AUTO_INCREMENT           | Уникальный идентификатор закладки |
| `user_id`    | INT      | FOREIGN KEY (`users.id`), NOT NULL    | Владелец закладки                 |
| `project_id` | INT      | FOREIGN KEY (`projects.id`), NOT NULL | Проект, добавленный в избранное   |
| `created_at` | DATETIME | NOT NULL                              | Дата и время добавления           |

### Связи между таблицами:
- Один **пользователь** может создать множество **проектов**
- Один **проект** может иметь множество **комментариев** и **оценок**
- Один **пользователь** может поставить одну **оценку** на проект
- Пользователи могут добавлять проекты в **закладки**
